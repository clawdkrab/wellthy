## PAYWALL PHILOSOPHY (CRITICAL)

A paywall should feel like:
> "This works. I want more."

Not:
> "I'm blocked."

### RULES
- Paywall only appears after value is experienced
- No urgency language
- No countdowns
- No "limited time"
- Always provide a free path

---

## PAYWALL TRIGGER (LOCKED)

Trigger paywall after:
- User completes Day 2
- User has:
  - a streak
  - earned credits
  - seen Collect screen

Never earlier.

## FREE VS PAID BEHAVIOUR (STRICT)

**Free users:**
- Can track movement
- Can earn credits
- Can build streaks
- Can unlock collectibles
- Can use trainers tab

**Paid users:**
- Faster progression
- More forgiveness
- Deeper engagement

**Paid never unlocks unrelated features.**

---

## COPY RULES (IMPORTANT)
- Never promise money
- Never imply income
- Never say "earn cash"
- Never reference crypto
- Never reference STEPN in-app

**STEPN is design inspiration only, not user-facing language.**

---

## EDGE CASES

**If user skips paywall:**
- Do nothing
- Do not nag immediately
- Allow them to continue

**If subscription fails:**
- Explain calmly
- Offer retry
- Allow continue free
